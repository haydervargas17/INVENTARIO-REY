# Frontend (implementación inicial)

Plantilla inicial con Vite + TypeScript + TailwindCSS y páginas base.

Pasos para ejecutar:

```bash
cd frontend
npm install
npm run dev
```

Variables de entorno (opcional):

- `VITE_API_BASE_URL` — URL base de la API (por defecto `/api/v1`).
